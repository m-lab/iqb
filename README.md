# Internet Quality Barometer (IQB)

The Internet Quality Barometer (IQB) is an open-source project initiated by [Measurement Lab (M-Lab)](https://www.measurementlab.net/).

IQB is motivated by the need to redefine how we measure and understand Internet performance to keep pace with evolving technological demands and user expectations. IQB is a comprehensive framework for collecting data and calculating a composite score, the ‚ÄúIQB Score‚Äù, which reflects the quality of Internet experience. IQB takes a more holistic approach than ‚Äúspeed tests‚Äù and evaluates Internet performance across various use cases (web browsing, video streaming, online gaming, etc.), each with its own specific network requirements (latency, throughput, etc.).

Read more about the IQB framework in:
- M-Lab's [blog post](https://www.measurementlab.net/blog/iqb/)
- The IQB framework [detailed report](https://www.measurementlab.net/publications/IQB_report_2025.pdf) and [executive summary](https://www.measurementlab.net/publications/IQB_executive_summary_2025.pdf)
- IQB [poster](https://arxiv.org/pdf/2509.19034) at ACM IMC 2025
